# Expense Split App - Product Requirements Document

## Overview
A comprehensive Android application for tracking shared expenses, managing debts, and simplifying group payments.

## Core Features
- **Add Groups & Friends:** Create specialized groups (Trip, Home, Office) and add participants.
- **Flexible Splitting:**
    - Equal/Unequal splits.
    - Split by percentage (%) or shares.
    - Itemization for detailed bills (e.g., restaurants).
- **Debt Management:**
    - Calculate total balances (individual and group).
    - Debt Simplification (minimum transactions to settle).
    - Payment Integrations (UPI, PayPal, Venmo).
- **Expense Tracking:**
    - Categorization (Food, Travel, Utilities, etc.).
    - Recurring expenses (Rent, Subscriptions).
    - Receipt scanning with OCR.
    - Offline mode with cloud sync.
- **Analytics & Tools:**
    - Spending totals and trends (Charts/Graphs).
    - Currency conversion (100+ currencies).
    - Transaction import (CSV, Bank statements).
    - Expense search and default split patterns.

## Target Audience
- Roommates sharing household bills.
- Travelers on group trips.
- Friends and colleagues managing shared social expenses.

## Visual Direction
- **Style:** Modern Android (Material You), clean, trust-focused.
- **Color Palette:** Deep indigo or forest green for financial stability, with vibrant accent colors for categories.
- **Typography:** Highly readable sans-serif (Inter or Roboto).
- **UX Focus:** Ease of data entry, clear balance visualizations, and "one-tap" settlement.
